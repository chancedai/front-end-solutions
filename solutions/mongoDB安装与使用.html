安装及连接

  1. 下载安装：http://www.mongodb.org/downloads
  2. 设置数据文件存放地址，如：G:\nodejs\MongoDB\bin>mongod.exe --dbpath G:nodejs\mongo_data
  3. 运行：mongod.exe ，提示类似等待连接 waiting for connections on port 27017，说明成功
  4. gbk文件需要在Gruntfile.js中设置 grunt.file.defaultEncoding = 'gbk';

手动创建MongoDB的windows服务

  1. 在G:nodejs\mongo_data下创建log文件夹
  2. 在G:\nodejs\MongoDB\mongod.cfg这个配置文件下配置logpath和dbpath，

    logpath=G:\nodejs\mongo_data\log\mongod.log

    dbpath=G:\nodejs\mongo_data\db

    可在cmd中运行来生成，如：echo logpath=G:\nodejs\mongo_data\log\mongod.log> "G:\nodejs\MongoDB\mongod.cfg" echo dbpath=G:\nodejs\mongo_data\db>> "G:\nodejs\MongoDB\mongod.cfg"
  3. 创建MongoDB服务：sc.exe create MongoDB binPath= "\"G:\nodejs\MongoDB\bin\mongod.exe\" --service --config=\"G:\nodejs\MongoDB\mongod.cfg\"" DisplayName= "MongoDB" start= "auto",运行后提示：[SC] CreateServer 成功
  4. 启动MongoDB服务：net start MongoDB
  5. 如要停止MongoDB服务：net stop MongoDB
  6. 如要删除MongoDB服务，先停止再执行：sc.exe delete MongoDB
