1. fiddler暂停： file>capture traffic 或 f12
  2. 不捕捉https： tools>fiddler options > https
  3. 允许其它设备访问：tools>fiddler options > connetions >allow remote;在其它设备上设置代理ip及端口号即可访问
  4. 模拟低网速：rules>performance>simulate modem speeds;如果要调整这个速度，可以rules>customize rules>搜索m_SimulateModem
  5. 在会话列表中，添加ip栏:rules>customize rules 在 static function Main()中添加一行 FiddlerObject.UI.lvSessions.AddBoundColumn("ServerIP", 120, "X-HostIP");
