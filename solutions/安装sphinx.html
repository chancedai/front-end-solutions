如何使用easy_install安装Sphinx 由于Sphinx是由python写的，所以要先安装python。 安装Sphinx步骤：
  • 1. 安装python： 见How to Install Python for Windows
  • 2. 利用easy_install来辅助安装python组件（easy_install本身也是python的一个组件），easy_install使得在python上安装组件变得非常方便。 如何安装easy_install: 首先下载ez_setup.py，然后直接使用python命令安装：python 下载目录\ez_setup.py setuptools
  • 3. 因为上一步中设置了PATH，所以可以直接使用easy_install命令来安装Sphinx：easy_install sphinx sphinx安装完成了之后就可以用它来制作各种文档了。
